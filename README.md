# m-wdl-archive

## 项目简介

m-wdl-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，而是作为一个通用容器，收录各类静态 HTML 文件，方便集中管理、版本回溯与分发。

## 目录说明

```
m-wdl-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面依赖的静态资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页或导航页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件独立存放，文件名应简洁且具有辨识度。
- `assets/`：按需组织，可包含子目录以区分不同页面的资源。
- `index.html`：若需要提供一个入口页面，可放置于此。

## 页面归档说明

- 所有归档页面均为静态 HTML，不含服务端逻辑。
- 页面之间相互独立，无依赖关系。
- 归档内容可能来源于不同时期、不同场景的 HTML 文档，仅作留存与展示用途。
- 不保证所有页面在最新浏览器下均完全兼容，但会尽量保持基础功能可用。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正归档页面。
- 新增页面请放入 `pages/` 目录，并确保资源路径正确。
- 不建议在仓库中存放过大的文件（如视频、高分辨率图片等），以免影响克隆与浏览体验。
- 本仓库的维护者会不定期整理目录结构，删除明显重复或无效的页面。
